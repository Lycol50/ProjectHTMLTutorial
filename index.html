Levels!


    There are 6 levels in this tutorial and every after level there will be quiz about a certain level.
Level 1-We will teach the first things that they need like a short history about html, HTML structure, Indentations and Title.
Level 2-Here we're going to tackle about heading tag - <h1> </h1>, paragraph tag - <p></p> and br tag -
so that they will learn how to make simple text only webpage.
Level 3-At this part we will teach them more about attribute name="value" like color attribute, size attribute and alignment
Level 4-Here we will teach them about list the Unordered list - <ul></ul> and Ordered list - <ol></ol> so that they can make or put a list in their webpage.
Level 5-Here we're going to teach them about image - how to insert, how to resize and what is the unit for size of an image inserted.
Level 6-Here we're going to study about linking or how to use the Anchor tag - so they can make their webpage connected to one another and will be called a website.
To start all the levels, Please Press "Start All" or just select a certain level below.
