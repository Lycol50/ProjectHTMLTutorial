HTML Attributes

The HTML Attributes

All HTML elements can have attributes. Attributes provide additional information about an element.

Attributes are always specified in the start tag. Attributes usually come in name/value pairs like:name="value". Example:


The Align Attribute

When this attribute this applied the element will position according to the code. there are 3 values in align attribute

  1. left
  2. center
  3. right

Example of Align Attribute

	 <html> 
	 <head> 
	  <title>Alignment Attribute </title>
	 </head>
	 <body>
	  <p align="right">im a paragraph.</p>
	 </body>
	 <html>

Output:


Color Attribute

When color attribute was used The Element will change it color according to the code.

Example of Color Attribute:

	 <html> 
	 <head> 
	  <title>Alignment Attribute </title>
	 </head>
	 <body>
	  <p style="color:red">Hello world</p>.
	 </body>
	 <html>

Output:


Width and Height Attribute

Width and Height Attribute is for resizing an element Width and Height Attribute can be used only on <img> and <embed> tag.

Example of Width and Height Attribute:

	 <html> 
	 <head> 
	  <title>Alignment Attribute </title>
	 </head>
	 <body>
	  <img src="img_girl.jpg" width="250" height="250">
	 </body>
	 <html>
